### Features

- ✔️ Cadastro de Profissionais
- ✔️ Alteração do Perfil
- ✔️ Filtragem de consultas por status
- ✔️ Confirmar, cancelar, rejeitar ou marcar consulta como realizada
- ✔️ Exclusão de conta
